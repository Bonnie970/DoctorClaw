# 常见药物相互作用参考 / Drug Interaction Reference

录入新处方时，对照此表检查同一成员的在用药物（medications 表 is_active=1）。

## 高风险组合 🔴

| 药物A | 药物B | 风险 | 建议 |
|-------|-------|------|------|
| 华法林 (warfarin) | 阿司匹林/NSAIDs | 出血风险显著增加 | 避免合用，必须合用需监测INR |
| 华法林 | 抗生素（头孢类、甲硝唑） | INR升高，出血风险 | 密切监测凝血 |
| 地高辛 (digoxin) | 胺碘酮 (amiodarone) | 地高辛浓度升高 | 减半地高辛剂量 |
| ACE抑制剂/ARB | 保钾利尿剂 (螺内酯) | 高钾血症 | 监测血钾 |
| 他汀类 | 大环内酯类抗生素 (红霉素/克拉霉素) | 横纹肌溶解风险 | 暂停他汀或换阿奇霉素 |
| 甲氨蝶呤 | NSAIDs | 甲氨蝶呤毒性增加 | 避免合用 |
| 锂盐 | NSAIDs/ACE抑制剂/利尿剂 | 锂浓度升高中毒 | 避免或密切监测锂浓度 |
| SSRIs | MAOIs | 5-HT综合征（致命） | 绝对禁忌 |
| 氯吡格雷 | 奥美拉唑 | 降低氯吡格雷疗效 | 换用泮托拉唑/雷贝拉唑 |

## 中等风险组合 🟡

| 药物A | 药物B | 风险 | 建议 |
|-------|-------|------|------|
| NSAIDs (布洛芬/双氯芬酸等) | 质子泵抑制剂(PPI) 适应症 | NSAIDs加重胃溃疡/GERD | 如有GERD/胃病史，需加PPI保护 |
| NSAIDs | 降压药 (ACE/ARB/利尿剂) | 降低降压效果、肾损伤 | 短期使用，监测血压肾功能 |
| 降糖药 (磺脲类) | 氟康唑/甲氧苄啶 | 低血糖风险 | 加强血糖监测 |
| 苯二氮卓类 | 阿片类 | 呼吸抑制 | 尽量避免合用 |
| 钙片 | 左甲状腺素 (优甲乐) | 降低甲状腺素吸收 | 间隔4小时服用 |
| 铁剂 | 左甲状腺素/喹诺酮/四环素 | 降低吸收 | 间隔2-4小时 |

## 常见慢性病+急性用药注意

### 高血压患者
- ⚠️ 避免含伪麻黄碱的感冒药（如新康泰克、白加黑）→ 升压
- ⚠️ NSAIDs长期使用 → 降低降压药效果

### 糖尿病患者
- ⚠️ 糖皮质激素 → 升血糖
- ⚠️ 含糖糖浆剂 → 注意含糖量

### GERD/胃溃疡患者
- ⚠️ NSAIDs → 加重胃损伤
- ⚠️ 阿司匹林 → 需评估风险收益
- ⚠️ 双磷酸盐 → 空腹服用，保持直立30分钟

### 痛风/高尿酸患者
- ⚠️ 噻嗪类利尿剂 → 升高尿酸
- ⚠️ 低剂量阿司匹林 → 可能升高尿酸
- ✅ 氯沙坦有轻度降尿酸作用

## 检查逻辑

录入新药时：
1. 查询该成员所有 `is_active=1` 的在用药物
2. 查询该成员 `chronic_conditions`（慢性病）
3. 对照上表检查是否存在冲突组合
4. 如有冲突，**立即警告用户**，说明风险和建议
5. 仍然录入药物（用户可能已知风险），但在 notes 中标注冲突提醒
