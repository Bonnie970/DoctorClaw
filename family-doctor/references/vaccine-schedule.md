## 二类疫苗（自费，建议接种）

| 疫苗 | 建议月龄 | 备注 |
|------|----------|------|
| 13价肺炎球菌疫苗 (PCV13) | 2, 4, 6月龄 + 12-15月龄加强 | 强烈建议 |
| 手足口病疫苗 (EV71) | 6月龄起，间隔1月共2剂 | 建议5岁前完成 |
| 轮状病毒疫苗 | 2月龄起 | 口服，根据品牌2-3剂 |
| 水痘疫苗 | 12月龄 + 4周岁 | 强烈建议 |
| 流感疫苗 | 6月龄起每年 | 首次2剂间隔4周 |
| Hib疫苗 | 2, 4, 6月龄 + 12-15月龄 | 如未打五联 |
| 五联疫苗 (DTaP-IPV-Hib) | 2, 3, 4月龄 + 18月龄 | 替代百白破+脊灰+Hib |

## 使用方法

1. 根据儿童出生日期计算当前月龄
2. 查询 vaccinations 表已接种记录
3. 对比上表找出应打但未打的疫苗
4. 提示家长待接种疫苗及建议时间
